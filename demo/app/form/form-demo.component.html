Default Form

If the ec-form element has no contents, the default form will be rendered, which is a list containing all fields and a submit button

Custom Content

As soon as you put something inside the ec-form element, the default content will be omitted. You can utilize the ec-input element to automatically render certain fields.

{{treeForm.group?.value | json}}

By Object

You can also generate a form by just passing an object to the value input. The form will analyze the object's properties and determine which form inputs should be shown by type.
