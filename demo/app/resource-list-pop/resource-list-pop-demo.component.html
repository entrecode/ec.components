resource-list-pop

This demo showcases the resource-list-pop, which is a dialog that contains a list of resources. All you have to provide is the api (ec.sdk Core conncetor) and a relation name to render the list. If you pass no config, the default config of ResourceConfig service will be used. The demo contains a tiny api-explorer that enables you to navigate through different apis.
